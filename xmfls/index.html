关于 「 使命精英2（内测版）」

「 使命精英2（内测版） 」是由谭淇元和叶锦达联合制作的项目

我们的目标是开发一款以射击游戏为地图场景的多端（手机、电脑、网页）游戏

目前第一阶段开发已完成、预计本学期发布

我们使用unity游戏引擎制作

包括游戏内的所有建模

并将逐步加入室内景观、天气系统等

如对游戏有任何修改建议请分享到七（1）班

旨在为所有七（1）班同学制作一款适合自己的射击游戏

分享你的创意，加入我们吧！

电脑以最高画质运行的最低配置

操作系统：64位 Windows 7 或更新的64位Windows操作系统

处理器：Intel Core i5-6600K / AMD FX-8350 或同级别的CPU

内存：8 GB RAM

显卡：NVIDIA GeForce GTX 970 / AMD Radeon R9 390 或同级别和以上的GPU

硬盘：3 GB 可用存储空间

手机以最高画质运行的最低配置

手机要求：有一部手机就行
